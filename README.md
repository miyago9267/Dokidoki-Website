# Dokidoki 心動不已告白日記網頁

## 簡介

委託

## 使用說明

### 安裝

1. 安裝node.js 18以上版本
2. 安裝yarn或npm
3. `yarn install` or `npm install`

### 開發

```bash=
yarn dev
```

### 輸出

```bash=
yarn build
```

接著將`.output`資料夾內容放到伺服器上，運行`node server/index.mjs`即可
或者可選用輸出後直接使用`yarn preview`運行預覽

### 預覽

```bash=
yarn preview
```
